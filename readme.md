## Getting start
    1. Clone this project
## Setting up project
    1. Go the folder where project is kept
    2. Run `npm init` on your command-line
## Development
    1. Go the folder forexample testing
    2. Run `webpack -w`
## Deployment
    1. Copy all src files in dist folder
    2. Move to your server ftp

Done :-D
